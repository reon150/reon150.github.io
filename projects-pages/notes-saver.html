Notes Saver Web App

Previous Next

En esta aplicación web se pueden almacenar notas que pueden escribir los usuarios que se han registrado en la aplicación.

Las notas se pueden editar / eliminar según sea necesario.

Para crear la base de datos use el enfoquee "code firts", para esto hice uso de Entity Framework Core

Para manejar el registro y login de usuarios use ASP.NET Core Identity

Tecnologías utilizadas

  • .NET Core

  • ASP.NET Core

  • SQL Server

  • Bootstrap 4
